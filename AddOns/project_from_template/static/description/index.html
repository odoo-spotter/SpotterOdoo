This module allows you to generate projects from templates previously created by the project manager. Enough wasting time creating and configuring projects with the same structure, instead, spend time and effort creating and configuring the necessary templates for all types of projects that your company manages and from which the projects will be generated.

Tasks within templates can be configured to automatically calculate the due date when the project is created based on the start date. When creating a new project from the wizard, it is possible to add and modify some values without affecting the original template.
